Class Detail
URLRequest(url)
A pixel is requested by dynamically generating an img element, which is appended to the document. After appending is complete, the img element is cleared from the document.
Author: christopher.sancho@adtech.com. 
var pixelRequest = new URLRequest('http://my.pixel-url.com');
pixelRequest.load();

// Alternate
var pixelRequest2 = new URLRequest();
pixelRequest2.url = 'http://my.pixel-url.com';
pixelRequest2.load();
